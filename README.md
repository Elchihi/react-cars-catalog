# React Cars Catalog

A responsive cars catalog application built with React and Vite.
The project allows users to browse cars, filter results, and sort data easily.

## Features

- Browse cars catalog
- Filter by brand
- Search by car name
- Min & Max price filter
- Sort by price and name
- Responsive design

## Tech Stack

- React
- Vite
- JavaScript
- CSS
